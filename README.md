# Calculator CLI App

## Project Overview

This project is a simple Command Line Interface (CLI) Calculator developed using Python. It allows users to perform basic arithmetic operations through an interactive menu.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Menu-driven interface
* Runs continuously until the user chooses to exit

##  Technologies Used

* Python 3
* Visual Studio Code (VS Code)
* Command Prompt / Terminal
* Git & GitHub

##  How to Run

1. Clone the repository:

   git clone https://github.com/rajraj59928/Calculator-CLI-App.git


2. Navigate to the project folder:

   cd Calculator-CLI-App
3. Run the program:
   python calculator.py
